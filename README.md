# Typinger-Quest: Antigravity Typing RPG

『Typinger-Z』のリスペクトをベースにした、浮遊感あふれるタイピングRPG。

## コンセプト
- **Antigravity Design**: 画面全体に漂う無重力感。
- **Neon Battle**: 暗闇に光るネオンエフェクトと粒子。
- **Magic System**: タイピングで溜めたコインで魔法を習得。

## 現状の機能 (MVP)
- **ライト特化バトル**: 3文字以下の単語を消すと爆発（範囲攻撃）が発生するキャラクター「ライト」を実装。
- **浮遊単語**: 単語がサイン波を描きながら無重力状態で浮遊。
- **コイン収集**: 敵を倒し、ショップ画面へ遷移。

## 操作方法
- 画面に表示される単語を入力してください。
- 最初の文字を入力すると、その単語にターゲットが固定されます。

## 開発環境
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Particle System (Canvas API)
